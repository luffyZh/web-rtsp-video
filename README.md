# web-rtsp-video

A Demo With Web Support RTSP Stream.

## 目录结构

```
————
 | -- node-rtsp     // node 转 rtsp demo
 | -- webrtc-rtsp   // webrtc 播放 rtsp demo
```

## 使用

### Node

```bash
cd node-rtsp && npm install
npm run start
```

### WebRtc

```bash
cd webrtc-rtsp
http-server
```
